# Temporary LaTeX Notes

A small GitHub Pages tool for drafting short LaTeX notes in the browser.

It provides a source editor, live KaTeX preview, line-level math error hints, local autosave, light and dark themes, snippet insertion, switchable keybindings (default/emacs/vim, including Alt+B and Alt+F word navigation), `.tex` import/export, rendered HTML export, PDF export, and PNG/SVG copy or fallback download. It is useful for quick text and equation checks before opening a full Overleaf project.

Note: browser image clipboard APIs require a secure context (`https://` or `http://localhost`). On unsupported browsers or blocked clipboard permissions, PNG/SVG copy falls back to file download.
